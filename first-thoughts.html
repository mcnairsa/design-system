Meta Design System

A design system for creating design systems.

The first version of Meta Design System [MDS] will use the Codecademy Personal Profile Website Portfolio Project as an example. MDS will define the process to be followed in creating design systems for use in other/upcoming Codecademy projects. MDS is both a template for and an example of the process.

Principles

The DS is a dynamic document. It is stable enough to be useful yet fluid enough to both lead and reflect the state of the organisation and its online presence.

Brand Vision and Values

The DS seeks to reflect the vision and values of the entity for whom it is created
