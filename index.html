Meta Design System

A design system for creating design systems.

The first version of Meta Design System [MDS] will use an (at this point ficitonal) blog, The Testing Times, as an example. MDS will define the process to be followed in creating design systems for use in other/upcoming Codecademy projects. MDS is both a template for and an example of the process.
